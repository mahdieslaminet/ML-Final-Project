# Futsal IRAN Dataset

## توضیح دیتاست
این دیتاست شامل اطلاعات بازیکنان فوتسال ایرانی است و برای تحلیل توانایی‌ها و استعدادهای فوتسال طراحی شده است.

## محتویات دیتاست
- مشخصات فیزیکی بازیکن: قد، وزن، ساختار بدنی
- مشخصات حرفه‌ای: سن شروع حرفه‌ای، تجربه بازی در تیم‌های مختلف
- مهارت‌های فنی: سرعت، چابکی، استقامت، پاس، دریبل، شوت

## نکات استفاده
- داده‌ها پیش‌پردازش شده و برای مدل‌های یادگیری ماشین آماده هستند.
- نمونه‌های ناقص بررسی و اصلاح شده‌اند.
- برای افزایش دقت تحلیل‌ها، توصیه می‌شود حجم داده‌ها را در آینده افزایش دهید.

## نحوه استفاده
1. بارگذاری فایل‌های دیتاست در محیط Python یا Jupyter Notebook
2. تحلیل داده‌ها یا استفاده در مدل‌های پیش‌بینی
